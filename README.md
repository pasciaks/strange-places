# strange-places
Strange Places - Submitted to Brackey's Jam 2026.1

## Here's a collection of instructions and development information for the game jam submission.

- Use mouse to interact with puzzles.
- Left mouse, right mouse, and zoom wheel are appropriate on different puzzles or not.
- You can save/load to and from the public puzzle - resulting in a collaborative effort to finish each puzzle.
- This collaboration for puzzles is for each puzzle and at varying sizes (easy, moderate, difficult, challenging).
